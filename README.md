# Raspberry-pico-call-routing
 Assembly programs for Raspberry Pi Pico simulating call routing
 
## Description
- Developed assembly programs for Raspberry Pi Pico to simulate call routing.
- Implemented binary input storage, LED/switch control, LCD display alternation, and register comparison logic.
- Strengthened skills in ARM Cortex-M0+ assembly programming, embedded systems, and debugging.

## Programs
- **Program 1:** Store 4-bit binary input
- **Program 2:** LED control with switch
- **Program 3:** Alternate 0/1 on LCD
- **Program 4:** Register comparison with LED output
